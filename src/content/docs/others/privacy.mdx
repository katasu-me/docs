arrow2nd（以下「運営者」といいます。）は、運営者の提供するkatasu.me（以下「本サービス」といいます。）における、ユーザーについての個人情報を含む利用者情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

## 1. 収集する利用者情報及び収集方法

本ポリシーにおいて、「利用者情報」とは、ユーザーの識別に係る情報、通信サービス上の行動履歴、その他ユーザーまたはユーザーの端末に関連して生成または蓄積された情報であって、本ポリシーに基づき運営者が収集するものを意味するものとします。本サービスにおいて運営者が収集する利用者情報は、その収集方法に応じて、以下のようなものとなります。

### （1）ユーザーからご提供いただく情報

本サービスを利用するために、または本サービスの利用を通じてユーザーからご提供いただく情報は以下のとおりです。

- メールアドレス
- ユーザー名
- プロフィール画像
- ユーザーがアップロードした画像ファイル
- 画像に付与されたタイトル、タグ等のメタデータ

### （2）ユーザーが本サービスの利用において、他のサービスと連携を許可することにより、当該他のサービスからご提供いただく情報

ユーザーが、本サービスを利用するにあたり、Googleアカウント等の外部サービスとの連携を許可した場合には、その許可の際にご同意いただいた内容に基づき、以下の情報を当該外部サービスから収集します。

- 当該外部サービスでユーザーが利用するID
- メールアドレス
- プロフィール画像
- その他当該外部サービスのプライバシー設定によりユーザーが連携先に開示を認めた情報

### （3）ユーザーが本サービスを利用するにあたって、運営者が収集する情報

運営者は、本サービスへのアクセス状況やそのご利用方法に関する情報を収集することがあります。これには以下の情報が含まれます。

- IPアドレス
- ユーザーエージェント（ブラウザ・デバイス情報）
- リファラ
- サーバーアクセスログに関する情報
- Cookie

## 2. 利用目的

本サービスのサービス提供にかかわる利用者情報の具体的な利用目的は以下のとおりです。

1. 本サービスに関する登録の受付、本人確認、ユーザー認証、セッション管理、ユーザー設定の記録等本サービスの提供、維持、保護及び改善のため
2. アップロードされた画像の保存、表示、サムネイル生成のため
3. アップロードされた画像が利用規約に違反していないかを確認するため
4. ユーザーのトラフィック測定及び行動測定のため
5. 本サービスに関するご案内、お問い合わせ等への対応のため
6. 本サービスに関する運営者の規約、ポリシー等（以下「規約等」といいます。）に違反する行為に対する対応のため
7. 本サービスに関する規約等の変更などを通知するため

## 3. 外部サービスへの情報提供

### 3-1. 画像モデレーション

本サービスでは、アップロードされた画像が利用規約に適合しているかを確認するため、OpenAI社のAPIを利用しています。

| 項目 | 内容 |
|------|------|
| 提供先 | OpenAI, LLC |
| 提供される情報 | アップロードされた画像 |
| 利用目的 | 画像の不適切なコンテンツの検出 |
| プライバシーポリシー | https://openai.com/policies/privacy-policy |

### 3-2. 認証サービス

本サービスでは、ソーシャルログイン機能を提供するため、以下の外部サービスと連携しています。

| 項目 | 内容 |
|------|------|
| 提供先 | Google LLC |
| 提供される情報 | 認証に必要なトークン情報 |
| 利用目的 | ユーザー認証 |
| プライバシーポリシー | https://policies.google.com/privacy |

### 3-3. インフラストラクチャ

本サービスは、Cloudflare社のサービスを利用して運営されています。

| 項目 | 内容 |
|------|------|
| 提供先 | Cloudflare, Inc. |
| 提供される情報 | 利用者情報全般（サーバー上で処理されるすべての情報） |
| 利用目的 | サービスのホスティング、データベース、ファイルストレージ |
| プライバシーポリシー | https://www.cloudflare.com/privacypolicy/ |

### 3-4. アクセス解析

本サービスでは、サービス改善のためにUmamiによるアクセス解析を行っています。Umamiはプライバシーに配慮した解析ツールであり、Cookieを使用せず、個人を特定する情報は収集しません。

| 項目 | 内容 |
|------|------|
| 提供先 | 運営者 |
| 提供される情報 | ページビュー、リファラ、ブラウザ情報、画面解像度、国・地域 |
| 利用目的 | サービスの利用状況の把握、改善 |
| プライバシーポリシー | https://umami.is/privacy |

### 3-5. 通報フォーム

本サービスでは、画像やユーザーの通報機能にTillyのフォームサービスを利用しています。

| 項目 | 内容 |
|------|------|
| 提供先 | Tilly |
| 提供される情報 | 通報時にユーザーが入力した情報 |
| 利用目的 | 通報の受付・処理 |

## 4. 第三者提供

### 4-1

運営者は、利用者情報のうち、個人情報については、あらかじめユーザーの同意を得ないで、第三者に提供しません。但し、次に掲げる必要があり第三者に提供する場合はこの限りではありません。

1. 運営者が利用目的の達成に必要な範囲内において個人情報の取扱いの全部または一部を委託する場合
2. 合併その他の事由による事業の承継に伴って個人情報が提供される場合
3. 第3項の定めに従って、外部サービスへ個人情報が提供される場合
4. その他、個人情報の保護に関する法律（以下「個人情報保護法」といいます。）その他の法令で認められる場合

### 4-2

運営者は、個人情報を第三者に提供したときは、記録の作成及び保存を行います。

### 4-3

運営者は、第三者から個人情報の提供を受けるに際しては、必要な確認を行い、当該確認にかかる記録の作成及び保存を行うものとします。

## 5. 安全管理措置

### 5-1

運営者は、利用者情報の漏えい、滅失または毀損の防止その他の利用者情報の安全管理のために、必要かつ適切な措置を講じています。

### 5-2

運営者は、利用者情報のうち個人情報の取扱いを第三者に委託する場合には、適切な事業者を選定し、個人情報の安全管理が図られるよう必要かつ適切な監督を行います。

### 5-3

運営者が講じる安全管理措置の具体的な内容については、本プライバシーポリシーに記載のお問合せ窓口にお問い合わせください。

## 6. データの保存期間

| データの種類 | 保存期間 |
|-------------|---------|
| ユーザーアカウント情報 | アカウント削除まで |
| アップロードされた画像 | ユーザーが削除するか、アカウント削除まで |
| サーバーログ | Cloudflareのデフォルト保持ポリシーに準拠 |

## 7. 個人情報の開示等の請求

### 7-1

運営者は、ユーザーから、個人情報保護法の定めに基づき保有個人データの利用目的の通知、保有個人データ又は第三者提供の記録の開示、保有個人データの内容の訂正・追加・削除、保有個人データの利用の停止・消去・第三者提供の停止のご請求（あわせて以下「個人情報の開示等の請求」といいます。）があった場合は、ユーザーご本人からのご請求であることを確認した上で、遅滞なくこれらに対応いたします。
ただし、個人情報保護法その他の法令により、運営者が開示の義務を負わない場合は、この限りではありません。

### 7-2

個人情報の開示等の請求の具体的な方法については、本プライバシーポリシーに記載のお問い合わせ窓口にお問い合わせください。

## 8. アカウント削除

ユーザーがアカウントを削除した場合、以下のデータは自動的に削除されます。

- ユーザーアカウント情報
- アップロードされたすべての画像
- 画像に付与されたタグ
- セッション情報
- 外部サービス連携情報

## 9. お問い合わせ窓口

本サービスに対するご意見、ご質問、苦情のお申出その他利用者情報の取扱いに関するお問い合わせは、下記の窓口までお願いいたします。

> contact@katasu.me

## 10. プライバシーポリシーの変更手続

運営者は、必要に応じて、本ポリシーを変更します。但し、法令上ユーザーの同意が必要となるような本ポリシーの変更を行う場合、変更後の本ポリシーは、運営者所定の方法で変更に同意したユーザーに対してのみ適用されるものとします。なお、運営者は、本ポリシーを変更する場合には、変更後の本ポリシーの施行時期及び内容を本サービスのウェブサイト上での表示その他の適切な方法により周知し、またはユーザーに通知します。
